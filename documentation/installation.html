Installation

Installing GENFIRE from Binaries

The easiest way to use GENFIRE is to download the appropriate precompiled binary for your operating system. These consist of just a single executable file that contains all of the code. There are 64-bit versions for Mac OS X 10.11, Windows 10, and Ubuntu 10.04. It's possible that the binaries may work on other versions of these operating systems, but as these are the three that I use at the time of writing this document they are the three that are guaranteed.

Installing GENFIRE as a Python package

To install GENFIRE as a python package, first get the source code from the git repository. Either download the code as a compressed file or clone the repository with

$ git clone "some address that I need to change"

From here the instructions depend on your OS

Mac OS X

Navigate to the source code directory

$ cd /path/to/GENFIRE

Install (most of the) dependencies using pip

$ pip install --upgrade pip
$ pip install -r requirements.txt

You will also need sip and PyQt4, which unfortunately cannot be installed with pip. They're easy to install using the package manager homebrew. The following commands will install homebrew and then use it to install sip and pyqt

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew install sip
$ brew install pyqt

You should now have all the dependencies necessary to run GENFIRE. To install it, make sure you are in the directory with setup.py and enter

$ python setup.py install

If everything worked, you can now launch the gui from the command line with

$ genfire

You can also use GENFIRE from python

$ python
$ import GENFIRE


Linux (Ubuntu 14.04)

Navigate to the source code directory

$ cd /path/to/GENFIRE

Install (most of the) dependencies using pip

$ pip install --upgrade pip
$ pip install -r requirements.txt

You will also need sip and PyQt4, this command should install both.

$ sudo apt-get install python-qt4

You should now have all the dependencies necessary to run GENFIRE. To install it, make sure you are in the directory with setup.py and enter

$ python setup.py install

If everything worked, you can now launch the gui from the command line with

$ genfire

You can also use GENFIRE from python

$ python
$ import GENFIRE


Windows 10

Open up a cmd prompt and navigate to the source directory

$ chdir C:\path\to\GENFIRE
$ import GENFIRE

Install (most of the) dependencies using pip

$ pip install --upgrade pip
$ pip install -r requirements.txt

You will also need sip and PyQt4.

$ sudo apt-get install python-qt4


Installation Troubleshooting

If you have trouble installing PyQt4 or sip, consult their documentation
If you have some problem with the "pip install -r requirements.txt" step, you can view the requirements.txt file to see the packages that are necessary, and try to install them one-by-one.
